# Changelog — Análise ICP: ajuste correto de Planos de ação

## Objetivo
Corrigir o visual dos Planos de ação e do topo das abas de análise sem alterar as funções já existentes.

## Ajustes
- Removido o nome replicado dentro das abas Análise ICP, Análise GDV e Resumo de Rotas.
- Reorganizado o topo das abas com ações à esquerda e indicadores à direita.
- Refeito o layout do modal Planos de ação com container opaco, borda e fundo corretos.
- Removido visual transparente/solto do modal.
- Corrigidos campos de texto e datas para usar o padrão de controle da plataforma, sem focus visual chamativo.
- Mantidos os fluxos de criar plano, editar objetivo, prazo, ações, anexos de imagem/PDF, preview e remoção.
